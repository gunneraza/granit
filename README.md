# Webpack Static

Config for static HTML


## Installation

Via Git

``` bash
$ git clone https://github.com/azabroflovski/webpack-static.git

```

``` bash
$ npm install

```


## Usage
Run dev server

``` bash
$ npm run dev

```

Build

``` bash
$ npm run build

```

Build server

``` bash
$ npm run build:server

```
